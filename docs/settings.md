# 3. 设置与个性化

## 3.1 基础设置

- **刷新频率**：可在 30秒至 300秒间自由调整，或输入自定义秒数，平衡数据实时性与流量消耗。
- **页面宽度**：PC 端支持通过滑块在 600px - 2000px 范围内调整页面宽度，适配不同分辨率的显示器。
- **组件开关**：可独立控制大盘指数、分组内搜索框在 PC/移动端的显示与隐藏。

## 3.2 排序个性化

- **形式切换**：排序入口支持“按钮”或“下拉单选”两种模式。
- **规则定制**：

- **拖拽排序**：调整不同排序规则（如估值、持仓收益、持有金额）的优先级。
- **别名编辑**：支持为排序规则设置别名（例如将“估算涨幅”重命名为“涨跌幅”），让界面更符合直觉。
- **启用开关**：按需开启或关闭特定的排序逻辑。

## 3.3 大盘指数自定义

- **自选指数**：从预设列表中勾选你关注的指数（如上证指数、纳斯达克、恒生指数等）。
- **显示排序**：通过拖拽调整指数在顶部的展示顺序。

## 3.4 PC 表格高级配置

- **列管理**：支持显示/隐藏特定数据列，或通过拖拽调整列的左右顺序。
- **列固定（冻结）**：支持将特定列（如基金名称）固定在左侧，方便在大表格中横向滚动查看。
- **全名展示**：一键切换是否展示完整的基金名称，避免长名称被截断。
- **跨分组同步**：支持将当前的表格配置（顺序、显示隐藏、固定状态）一键同步到其他基金分组，无需重复设置。

## 3.5 移动端表格高级配置

- **个性化设置抽屉**：点击列表头左侧的“设置”图标，可唤起底部抽屉进行配置：

- **展示完整基金名称**：由于屏幕宽度限制，默认会截断长基金名，开启此项可展示全名（配合自动缩放字体）。
- **表头展示与排序**：支持隐藏不关注的列（如近1年收益、持仓成本等），并可通过拖拽调整各列的显示顺序。
- **跨分组同步**：同样支持将当前分组的表格布局配置一键同步至其他所有分组。

- **编辑与批量操作**：

- **进入编辑模式**：在默认排序模式下，点击表头的“铅笔”图标或**长按**任意基金行，即可进入批量编辑状态。
- **批量移动/删除**：在编辑模式下，可勾选多只基金，一键移动到其他分组或进行批量删除。
- **快速置顶**：编辑模式提供“移到最前”按钮，方便快速调整核心关注基金的顺序。
- **手动拖拽排序**：在编辑模式下，可通过右侧的拖拽手柄自由调整基金的上下位置。

## 3.6 数据管理

- **导入导出**：支持将所有配置与数据导出为 JSON 文件进行本地备份，或通过导入功能实现数据迁移。
- **防息屏模式**：提供“保持屏幕常亮”功能，适合将旧手机/平板作为行情监控副屏。
- **主题切换**：支持亮色/暗色模式，并可与系统主题自动同步。
