# 2. 高级特性

## 2.1 截图/OCR 导入

- **批量添加**：如果你有大量持仓，只需上传支付宝、天天基金等平台的持仓截图，系统将利用 OCR 技术自动识别基金代码、份额及成本，实现“一键搬家”。

## 2.2 自动定投计划

- **定投模拟**：支持设置日、周、月等周期性的定投计划，系统将自动记录定投流水，助你坚持长期投资。

## 2.3 分组管理

- **场景化分组**：支持创建多个分组（如“我的实盘”、“自选观察”等），不同分组独立统计收益，管理更清晰。

## 2.4 云端同步

- **多端联动**：登录邮箱账号后，数据可跨设备实时同步（PC、手机、平板），再也不怕更换设备。注意：同一时间段仅允许一台活跃设备往云端同步。当同步时出现“其它设备登录提示”弹框，点击“立即接管”可将当前设备设置为活跃设备。
